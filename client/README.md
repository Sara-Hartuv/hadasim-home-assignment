# Client

## תיאור
אפליקציית React להצגת:
- מורות ותלמידות
- חיפוש וסינון
- מפה עם מיקומים

---

## טכנולוגיות
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- React Leaflet

---

## פיצ'רים

### מורה:
- צפייה בתלמידות הכיתה
- הוספת תלמידות לכיתה של המורה המחוברת בלבד
- חיפוש תלמידה
- צפייה במפה
- תלמידות הרחוקות מהמורה מסומנות באדום + שם התלמידה

### מנהל:
- צפייה בכל המורות
- צפייה בכל התלמידות
- הוספה תלמידות
- הוספת מורות
- חיפוש תלמידות
- חיפוש מורות

---

## מפה

- הצגת מיקומים בזמן אמת
- אייקון במיקום המורה
- אייקון במיקום התלמידה עם שמה 
- אייקון אדום = רחוקה
